# Exercícios de C

Exercícios feitos durante o curso de C na Alura.

## Arrays
- nomes.c — lê 10 nomes e imprime só os que têm mais de 4 letras
- palavra_casa.c — dado 3 chutes, mostra as letras acertadas e _ nas que não foram

## Ponteiros
- exe1.c — declarar ponteiro e alterar valor de variável
- exe2.c — trocar valores de duas variáveis com ponteiros
- exe3.c — passar ponteiro pra função e dobrar valor
- exe4.c — encontrar maior valor de vetor com aritmética de ponteiros
- exe5.c — inverter string com dois ponteiros
